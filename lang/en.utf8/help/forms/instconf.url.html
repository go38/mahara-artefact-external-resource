Url

Enter the url of an external ressource. Any page that support one of the protocols listed below is supported. Note that you may need to refresh the page to see the result of some urls.

oEmbed

YouTube, Dailymotion, Slideshare, Photobucket, Flick, Hulu, Vimeo, Embedly, Viddler, Revision3, iFixit, Poll Everywhere, SmugMug, Soundcloud, more.

Open Graph Protocol/Facebook

Google Books, National Geographic, My Space, wordpress.com, Digg, IMDb, All Recipes, CNN, dotSUB, more.

Web Resource

Images, RSS, PDF, Word, PowerPoint, Excel, PostScript, more.

Google Application

Google Gadget, Google Books, Google Map.

Miscellaneous

Scratch.

Web Page

The defaut behaviour is to display a link to a web page with it's title, description and a default icon.
