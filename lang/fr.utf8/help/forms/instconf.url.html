Url

Entrer l'url d'une ressource qui supporte l'un des protocoles listés ci-dessous. Noter qu'il peut être nécessaire de rafraichir la page pour voir le résultat.

oEmbed

YouTube, Dailymotion, Slideshare, Photobucket, Flick, Hulu, Vimeo, Embedly, Viddler, Revision3, iFixit, Poll Everywhere, SmugMug, Soundcloud, plus.

Open Graph Protocol/Facebook

Google Books, National Geographic, My Space, wordpress.com, Digg, IMDb, All Recipes, CNN, dotSUB, plus.

Ressource Web

Images, RSS, PDF, Word, PowerPoint, Excel, PostScript, plus.

Google Application

Google Gadget, Google Books, Google Map.

Divers

Scratch.

Page Web

Le comportement par défaut est d'afficher le titre de la page, sa description s'il existe avec une icône par défaut.
